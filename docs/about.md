# About

[Studio B](https://b.studio) is a management consulting firm that builds software and ships professional services for mid-market Acumatica shops. We build two products on top of this SDK: **[Bolt](https://bolt.b.studio)**, a Continuous Goods WMS resold by Acumatica VARs, and **[AcuOps](https://acuops.com)**, an Acumatica DevOps platform sold directly to VARs. Both products talk to Acumatica, HubSpot, Railway, Slack, Microsoft 365, and a handful of other systems every minute of every day — `@studio-b-ai/clients` is the typed, tested surface we use for all of it. Publishing it as its own npm package (with sigstore provenance) means VARs carrying Bolt and extension authors building against AcuOps can import the exact same clients we run in production, rather than reverse-engineering calls from documentation. If you're reading this because you landed here from Bolt or AcuOps — welcome, and the [README](../README.md) is where to start.
